依存と自立 — プレゼンテーション

【使い方】
1. このフォルダをそのまま保持
2. index.html をブラウザで開く
3. スクロールで進行

【フォルダ構成】
presentation_package/
├── index.html             プレゼン本体
├── assets/
│   ├── tbc.png            To Be Continued画像(黒背景)
│   └── photos/            旅の写真
│       ├── india.jpg          インド食堂(3人)
│       ├── india1.jpg         インド遺跡(4人)
│       ├── turkey.jpg         トルコ夜の店(3人)
│       ├── france.jpg         フランス、コルマール
│       ├── brasil.png         リオのビーチ(4人)
│       ├── paraguai.png       パラグアイ(4人)
│       ├── uyuni.png          ウユニ塩湖
│       └── hero_machupichu.png  マチュピチュ(大型表示用)
└── README.txt

【注意】
- index.html と assets/ は同じ場所に置く
- ファイル名・フォルダ名は変えない
- 写真差替えは同じファイル名で上書き

【ブラウザ】
Chrome / Edge / Safari 最新版、F11全画面推奨
